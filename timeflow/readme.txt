TIMEFLOW ALPHA RELEASE

Welcome to the alpha release of TimeFlow, 
an analytical timeline application.

If you just want to install and use TimeFlow, please see:
http://wiki.github.com/FlowingMedia/TimeFlow/
which contains links to documentation and important
license information.

If you're interested in helping with development, see the Github developer
guide, linked from the URL above.
